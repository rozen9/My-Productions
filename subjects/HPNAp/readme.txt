/*
・課題で作成したニューラルネットワークに関する
シミュレーション用のプログラムです．
・このネットワークは近似的に自己想起連想記憶を実現させたため，
3パターン以上を設定すると上手く動作しません．
・今後プログラムの編集を行う予定です．
*/

・仕様
既存の0~9までの数字のパターンを2つ選び，Hopfield ネットワークにそのうち1つのパターンを
少し変化させたものを記憶させ，初期状態に戻るか確認する
(つまりその2つのパターンが平衡状態になっているかどうか調べる).
上記をHopfield ネットワークを用いて実装．

具体的には

・初期状態を生成するための関数
・パターンとして使用するデータをまとめる関数
・結合の強さを設定するための関数
・非同期的状態変化を行うための関数

を実装する．
初期状態はあるパターンを一つ選んで，確率0.1 で0，1 を反転させたものとする．
結合の強さは近似的な自己想起的連想記憶を用いて実現する．
